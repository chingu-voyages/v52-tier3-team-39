# Rayvolution

Client application built with Next.js

## Major Dependencies

- React 19
- Next 15
- ESLint
- Tailwind CSS

## Getting Started

#### Environment variables

Create a `.env` file in the root of the `client` directory:

```
SERVER_URL=http://localhost:4000/
```

#### Run app in dev mode

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
